Welcome to the AI-powered Facebook Post Automation tool! This project is built to help users automatically generate and schedule Facebook image posts with custom prompts and optional captions.

The goal is to automate the process of generating, reviewing, and publishing Facebook image posts so creators and marketers can save time, maintain consistency, and focus more on content strategy rather than manual work.

‚ú® Features

  • User authentication via email/password (social login coming soon)

  • Auto-create batches with filters like ratio, quality, and categories
  • Review AI-generated images and delete unwanted ones before posting
  • Schedule posts in a custom date/time range (with 30 min interval)
  • Each post includes only one image and optional text content
  • Cloudinary is used to store generated images
  • MongoDB stores users, prompts, images, posts, and schedules
  • Email verification and password reset with OTP
